# Portfolio Site

This folder contains a one-page portfolio draft for Candace Gillhoolley.

## Files
- `index.html`: Complete responsive portfolio page with embedded CSS.
- `resume.html`: Interactive resume page generated from `Candace_Gillhoolley_Resume.md`.
- `Candace_Gillhoolley_Resume.md`: Source resume content.

## How to use
1. Open `index.html` in a browser.
2. Replace placeholder items in:
   - Featured Articles
   - Featured Podcast Episodes
   - Results & Testimonials
3. Update links and wording as needed.
