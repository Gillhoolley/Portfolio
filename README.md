# Portfolio Site

This folder contains a static portfolio site for Candace Gillhoolley.

## Files
- `index.html`: Main responsive portfolio page with embedded CSS and JavaScript.
- `resume.html`: Interactive resume page generated from `Candace_Gillhoolley_Resume.md`.
- `Candace_Gillhoolley_Resume.md`: Source resume content.
- `candace.jpg`: Portrait used by the hero, favicon, and social preview metadata.
- `*.png`: Local book cover assets used in the work explorer.

## How to use
Open `index.html` in a browser. No build step or local server is required.

## Notes
- Typography uses `Fraunces` and `Inter` via Google Fonts.
- Colors and layout are defined with CSS variables in `:root`.
- The work explorer filters books, articles, podcasts, and YouTube Shorts.
- The page includes Open Graph and Twitter Card metadata for cleaner link previews.
